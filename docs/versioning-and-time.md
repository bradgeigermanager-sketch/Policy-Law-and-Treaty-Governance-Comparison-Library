# Versioning and Time

Each instrument (treaty, law, policy) has versioned instances:

- `treaty_version.*`
- `law_version.*`
- `policy_version.*`

Effective intervals:

- `effective_from` (inclusive)
- `effective_to` (exclusive or null)

Time-sliced impact:

- `unified_participant_impact_timeslice_engine`
- `report.unified_participant_impact_timeslice.*`
