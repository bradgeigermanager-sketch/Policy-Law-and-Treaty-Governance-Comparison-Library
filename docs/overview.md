# Governance OS – Overview

Governance OS is a schema-driven system for modeling, simulating, and optimizing the impact of treaties, laws, and policies on participants across jurisdictions and time.

Core ideas:

- Instruments: treaties, laws, policies (with versions)
- Scopes: jurisdictions, parties, participants
- Impacts: obligations, rights, limitations, conflicts, risks
- Engines: per-layer impact, cross-layer unification, optimization, harmonization, simulation
- Time: version-aware, time-sliced impact views
- UI: dashboards and panels for human navigation
